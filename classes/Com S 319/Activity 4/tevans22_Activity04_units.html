Tyler N Evans (Vertical).

"font-size:5vh;" in div sets the div font size to be 5% of the viewport height
"font-size:200%;" in h1 sets the h1 font size to be 200% of the div's font size
These font size settings are useful because they ensure that the font size always scales with and matches the viewport size.
